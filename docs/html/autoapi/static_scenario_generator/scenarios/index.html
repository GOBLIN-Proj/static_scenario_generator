static_scenario_generator.scenarios

A module for generating scenario dataframes from configuration files.

This module supports loading scenarios from JSON or CSV files, processing them into pandas DataFrames based on predefined common and animal-specific columns and systems. It leverages the DataManager for accessing data schemas and configurations.

class static_scenario_generator.scenarios.ScenarioGeneration[source]

A class to generate scenario dataframes from configuration files.

This class supports loading and processing scenarios from either JSON or CSV files, transforming them into pandas DataFrames according to predefined column schemas and system configurations. It utilizes the DataManager class to access these schemas and configurations.
